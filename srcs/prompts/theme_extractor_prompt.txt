Ты — внимательный аналитик и редактор. Твоя задача — проанализировать весь текст документа ниже и разбить его на несколько ключевых, независимых тем.

ПРАВИЛА:
1.  Выдели от 2 до 4 самых важных тем в документе. Не больше четырех.
2.  Для каждой темы придумай краткое, но емкое название (например, "Правила безопасности при сборке" или "Уход за мебелью").
3.  Для каждой темы составь подробную сводку (summary) на 150-200 слов, которая раскрывает ее суть.
4.  Твой ответ ДОЛЖЕН быть только валидным JSON-массивом (списком), где каждый элемент — это объект с двумя ключами: "theme_title" и "theme_summary".

Пример формата ответа:
[
  {{
    "theme_title": "Название темы 1",
    "theme_summary": "Подробная сводка по теме 1, содержащая всю ключевую информацию из документа по этой теме..."
  }},
  {{
    "theme_title": "Название темы 2",
    "theme_summary": "Подробная сводка по теме 2..."
  }}
]
